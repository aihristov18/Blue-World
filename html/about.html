Want to know about us?

♦ This project was made by a small group of ninthgraders wishing to help the World and save it from pollution.

♦ We want to achieve this goal by using this project to inform people about the dangers of pollution and unite the people of the World against this global problem.

♦ We have an interactive map which upon clicking a country displays live information about several things related to this country for example its' GDP and population. It is quite simple to use as well. With the left click on the mouse you select a continent then a country and with the right you zoom out when finished.
